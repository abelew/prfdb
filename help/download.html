The download interface currently provides a way to quickly download all sequences for a particular genome. Currently this only provides Saccharomyces cerevisiae sequences.

Downloading a sequence It is also possible to download individual sequences from the PRFdb as fasta files. When viewing sequences at the 'genome' view, it is possible to download the entire sequence as a fasta file. It is also possible to download smaller subsequences while viewing the 'detailed' view of a particular slippery site.
