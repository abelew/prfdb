Candidate cPRF signals are available as a batch download from either of the two text files below. Each file is prefaced with the SQL used to generate the results, for clarity. If you desire direct access to the PRFdb via a MySQL client, then please contact Dr. Dinman (dinman@umd.edu). Right-click and choose "Save as..." and then open these up in your favorite plain-text editor.

cPRF_niceformat.txt

This is probably the document that most people will want. It's formatted linearly, so that each record has it's own section. This document can be open in Word without too much trouble.(1.2 MB)

 

cPRF_cols.txt

 

Column formatted document. Becuase the sequence and basepaired columns are so wide, this document is really not intended for use with Word or even Excel. Open this with a plain-text editor and then use Perl or someother scripting language to parse the data.(780 Kb)

 

The prfdb

This archive contains the code of the PRFdb. It includes the source for the secondary structure algorithms used to generate the database as well as a default configuration file.
